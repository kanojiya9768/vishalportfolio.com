HTML CHEATSHEET

  1. Html BoilerPlate :-This is the basic template or barebone structure of HTML. 
    
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
                        
</body>
</html>
  2. Headings :- There are Three Headings From h1 to h6 in increasing order 
    
<h1>Heading 1</h6>
<h2>Heading 2</h6>
<h3>Heading 3</h6>
<h4>Heading 4</h6>
<h5>Heading 5</h6>
<h6>Heading 6</h6>
  3. Container Tags :- Container tags are the tags that contain some data such as text, image, etc. There are several container tags in HTML.

    div tag :- 

    <div> This is div block </div>
  4. Span Tag :- span is a container for inline content 
    <span> This is span block </span>
  5. Paragraph :- 
    <p> This is Paragraph </p>
  6. Pre Tag :- pre tag represents pre-formatted text 
    <pre> This is pre tag </pre>
  7. Code Tag :- code tag is used to represent source codes 
    <code> import python </code>
  8. Text Formating :- Text formatting tags are used to format text or data of HTML documents. You can do certain things like creating italic, bold, strong text to make your document look more attractive and understandable.

    Bold tag :- 

    <b>I'm bold text</b>
  9. Strong Tag :- 
    <strong>I'm strong text</strong>
  10. Italic Tag :- 
    <i>I'm Italic text</i>
  11. Emphasized Tag :- 
    <em>I'm emphasized text</em>
  12. Subscript Tag :- 
    <sub>I'm bold text</sub>
  13. Lists :- Lists can be either numerical, alphabetic, bullet, or other symbols. You can specify list type and list items in HTML for the clean document. 
    <ol> tag
  14. Ordered List :- Ordered list starts with <ol> tag and each list item starts with <li> tag 
    <ol>
<li>Data 1</li>
<li>Data 2</li>
<li>Data 3</li>
</ol>
  15. Unordered List :- Unordered list starts with <ul> tag and each list item starts with <li> tag 
    <ul>  
<li>Your Data</li>
<li>Your Data</li>
</ul>
  16. Media :- Media is anything that is present in digital form such as image, video, audio, etc.

    Audio tag :- 

    <audio> tag
  17. It is used to embed sound content in the document. 
    <audio controls>
<source src="demo.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>
  18. Image Tag :- It is used to embed or import image in a webpage. 
    <img src="Source_of_image" alt="Alternate text">
  19. Video Tag :- It is used to embed video in the webpage. 
    <video width="480" height="320" controls>
<source src="demo_move.mp4" type="video/mp4">
Your browser does not support the video tag.
</video>
  20. Table :- A table is a collection of rows and columns. It is used to represent data in tabular form.

    Table Structure :-

    <table>
<caption>Demo Table</caption>
<thead>
<tr>
<th>Column1</th>
<th colspan="2">Column2</th>
</tr>
</thead>
<tbody>
<tr>
<td>Data1</td>
<td>Data2</td>
<td>Data2</td>
</tr>
<tr>
<td>Data1</td>
<td>Data2</td>
<td>Data2</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>&nbsp;</td>
<td>Data</td>
<td>Data</td>
</tr>
</tfoot>
</table>
  21. Link :- Links are clickable text that can redirect you to some other page. 
    <a> tag
  22. <a> or anchor tag defines a hyperlink. 
    <a href="https://www.google.com/">Visit Google.com!</a>

  24. Characters & Symbols :- Some symbols are not directly present on the keyboard, but there are some ways to use them in HTML documents. We can display them either by entity name, decimal, or hexadecimal value.

    Copyright Symbol (©)

    &copy;

    Less than (<)

    &lt;

    Greater than (>)

    &gt;

    Ampersand (&)

    &amp;

    Dollar ($)

    &dollar;
  25. Semantic Elements :- Semantic elements are those elements that are self describable, i.e., from their name itself, you can understand their meaning. 
    <section> tag
    It defines a section in the document
    <section>This is a section</section>
    <article> tag :- It represents self-contained content
    <article> Enter your data here </article>
    <aside> tag :- IIt is used to place content in the sidebar
    <aside> Your data </aside>
